Quick Reference

Character Classes

. Any character (except newline)
\d Digit [0-9]
\w Word character [a-zA-Z0-9_]
\s Whitespace
[abc] Character set
[^abc] Negated set

Quantifiers

* 0 or more
+ 1 or more
? 0 or 1
{n} Exactly n
{n,} n or more
{n,m} Between n and m

Anchors

^ Start of string/line
$ End of string/line
\b Word boundary

Groups

(abc) Capture group
(?:abc) Non-capturing group
(?<name>abc) Named group
a|b Alternation

RE2/Go Limitations

\1 Backreferences - NOT supported
(?=...) Lookahead - NOT supported
(?!...) Negative lookahead - NOT supported
(?<=...) Lookbehind - NOT supported
